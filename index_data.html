The data

The data set was mainly gathered from three different sources. Some statistics about Iceland's population and others were obtained from Statistic Iceland.

Car accident data

Obtained from the Icelandic Transport Authority. In this data set we have all of the accidents that have occured in Iceland from the year 2007-2020. This includes accident type and the location of the accident.
